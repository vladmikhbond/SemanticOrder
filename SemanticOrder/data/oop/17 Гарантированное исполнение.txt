@1 17 Гарантированное исполнение

@3 Для гарантированного исполнения некоторого кода его помещают в блок finally, следующий за блоком try.  
{{
try {
     // рискованный код
}
finally
{
     // обязательный код
}  
}}

@3 Пример. 
{{
TextWriter writer = null;
try
{
    writer = new StreamWriter("d:\\111.txt");
    foreach (string s in ss)
        writer.WriteLine(s);
}
finally
{
    writer.Close();
}  
}}

@2 Переходы в блоке try

@3 Если рискованный код не вызвал исключения, то выполнение переходов откладывается до окончания исполнения обязательного кода.
{{
try {
     // рискованный код
     return …
     throw …
     goto …
}
catch 
{
     // код обработки
}
finally
{
     // обязательный код
}     
}}

@2 Блок using

@3 Гарантированное высвобождение ресурсов обеспечивает блок using. 
{{
using (TextWriter writer = new StreamWriter("books.txt"))
{
    foreach (string s in ss) {
        writer.WriteLine(s);
   }
}   
}}

@@ @3 Блок using возможен лишь для объектов, реализующих интерфейс IDisposable.
