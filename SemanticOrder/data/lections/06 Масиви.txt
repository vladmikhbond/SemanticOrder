@2 Масиви

@3 Конструктором масиву є функція Array() або літерал.
{{
let a = new Array("one", "two");
let b = ["one", "two"];
let c = new Array(100500).fill(0);
}}

@3 Хоча масив є об'єктом і може мати будь-які властивості, 
він задуманий як колекція властивостей із цілими ключами: 0, 1, 2, ... 
Ключи властивостей масиву називають індексами, а значення властивостей - елементами масиву.

@3 Властивість length має значення на 1 більше максимального індексу масиву.
{{
let a = ["one", "two"];
a[10] = "ten";

console.log(a)
}}

@3 {Примітка.} Властивість length можна змінювати.

@3 Для перебору елементів масиву застосовують цикл for
{{
let a = ["Яблуко", "Апельсин", "Груша"];

for (let i = 0; i < arr.length; i++)
  console.log(a[i]);
}}

@3 Цикл for-in теж дозволяє пройти масив, але він перебере не лише числові властивості.
{{
let a = ["Яблуко", "Апельсин", "Груша"];
a.prop = "Кролик";
for (var key in a) {
    console.log(a[key]);
}
}}

@3 {Примітка}. Цикл for-in у браузерах працює помітно повільніше за цикл for.

@2 Методи масивів

@3 Методи pop і push додають та видаляють елементи з кінця масиву.
Методи shift та unshift роблять те саме з початку.

@3 Метод slice(begin, end)
копіює ділянку масиву від індексу begin включно до індексу end виключно. Початковий масив у не змінюється.
від'ємний індекс означає відлік з кінця (-i = length – i).

@3 Метод splice(index[, deleteCount, elem1, ..., elemN])
видаляє deleteCount елементів, починаючи з позиції index, і сюди вставляє нові елементи elem1, ..., elemN.
Від'ємний індекс трактується, як у методі slice.

@3 Метод sort() сортує масив на місці.
Має необов'язковий параметр – функцію-компаратор.
{{
function compareNumeric(a, b) {
  return a - b;
}

var arr = [1, 2, 15];
arr.sort(compareNumeric);
alert(arr); // 1, 2, 15
}}
Якщо компаратор не заданий, елементи сортуються як рядки.

@3 Інші методи:
reverse
concat
indexOf / lastIndexOf

@2 Багатовимірні масиви

@3 Багатовимірних масивів не існує, але елементи масиву можуть бути масивами.
Такі масиви створюють за допомогою циклів.
Приклад - масив з m "рядків" і n "стовпців".
{{
    const table = new Array(m);
    for (let i = 0; i < table.length; i++)
        table[i] = new Array(n);
}}
