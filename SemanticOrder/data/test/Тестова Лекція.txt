@2 Цикл while
@@ while|цикл+

Щоб виконати команду багаторазово, її потрібно вкласти в команду while 
(вкладена команда виконується 0 або n разів).

Знову перевіряється умова і т. д.
Коли умова стає неправдивою, команда while завершується.

@3 Вивести на консоль числа від 0 до 9.
{{
let i = 0;
while (i < 10)
{
   console.log (i);
   i = i + 1;
}
}}
