@1 08 Функції

@2 Функції
@@ функці+|function|return

@3 Функція - фундаментальне поняття не тільки математики, але і програмування.

@3 Функція задає однозначну відповідність між двома множинами.
[[Function.png]]

@3 В математиці відповідність можна встановити по-різному: 
виразом, 
таблицею, 
графіком 
або іншим чином.

@3 У програмуванні відповідність можна встановити тільки алгоритмом, 
який обчислює значення функції по її аргументах.

@3 Для цього існує спеціальна команда - function.
Наприклад, визначимо функцію, яка виконує цілочисельне ділення x на y.
{{
function div(x, y)
{
    let r = x % y;
    return (x - r) / y;
}
}}

@3 Команда return сповіщає інтерпретатор, що є результатом роботи функції.

@3 Один раз визначивши функцію, ми можемо скористатися нею скільки завгодно раз.
{{
a = div(10, 6);
b = div(33, 4);
...
}}

@3 Вираз div(x, y) можна розуміти як ще одну команду JS,
якою можна користатися в області видимості функції div.

@2 Приклади функцій
@@ факторіал+|gcd

@3 Функція max(x, y) повертає більше з двох чисел x і y.
{{
function max(x, y)
{
    let z;
    if (x > y)
      z = x;
    else
      z = y;
    return z;
}
}}

@3 Функція обчислення факторіала.
n! = 1 * 2 * 3 * ... * n.
{{
function factorial(n)
{
   let f = 1;
   while (n > 0) {
       f *= n;
       n -= 1;
   }
   return f;
}
}}

@3 Функція знаходження найбільшого спільного дільника.
{{
function gcd(a, b)
{
    while (b > 0)
    {
        let r = a % b;
        a = b;
        b = r;
    }
    return a;
}
}}

@2 Підсумки

@3 В програмуванні функція - це завжди обчислювальна процедура.
@3 Визначення функції можна сприймати як доповнення мови локальної командою.
