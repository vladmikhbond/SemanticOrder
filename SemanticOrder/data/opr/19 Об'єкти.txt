@1 19 Об'єкти

@2 Об'єкти
@@ об'єкт+|властивост|Object

@3 Об'єкт можна розглядати як контейнер для змінних (їх називають властивостями об'єкта).
{{
let man = { name: "Igor", age: 17, height: 185 }
}}

@3 Виклик конструктора new Object() створює об'єкт без властивостей.
Приклад: створити об'єкт і надати йому властивості: year, month і day.
{{
let date = new Object();
date.year = 2021;
date.month = "жовтень";
date.day = 19;
}}

@3 Літерали об'єктів - теж конструктори.
{{
let date = {year: 2021, month: "жовтень", day: 19};
let empty = {};
}}

@2 Властивості
@-

@3 Звертатися до властивостей об'єкта можна двома способами:
{{
data.year
або
data["year"]
}}

@3 Те ж саме в літералах:
{{
{year: 2021, month: "жовтень", day: 19};
або
{"year": 2021, "month": "жовтень", "day": 19};
}}

@3 Якщо назва властивості не є ім'ям з точки зору JS, 
перший спосіб виключається, залишається тільки другий.
{{
let date = {"рік": 2021 року, "місяць": "жовтень", "день місяця": 19};
alert (date ["день місяця"]);
}}

@3 Переконатися в існуванні властивості у об'єкта можна, порівнявши властивість з undefined.
{{
date.color === undefined
або
date["color"] === undefined
}}

@2 Функціональні властивості (методи)
@@ метод+|delete|оператор

@3 Властивості об'єкта можуть бути не тільки даними, але і функціями.
{{
let obj = {
  x: 5,
  sum: function (a, b) {return a + b},
  g: Math.abs}
}}

@3 JS не вбачає великої різниці між даними і функціями.
Функції - це дані, які до того ж можна і викликати.

@3 JS дозволяє додавати і видаляти властивості у будь-який час існування об'єкта.

@3 {Зауваження}. Додавати властивості можна не тільки екземплярам типу Object,
але і іншим змінним об'єктам - масивам, функціям.

@3 Видаляє властивість оператор delete.
{{
let rect = {width: 10, height: 12, color: "red"};
delete rect.color;
}}

@2 Прибирання сміття
@@ смітт+

@3 Оператор delete розриває зв'язок між ім'ям і значенням.
Якщо якесь значення позбавляється всіх своїх імен, воно буде підібрано збирачем сміття.

@3 Збирач сміття - служба інтерпретатора, яка звільняє пам'ять від тих об'єктів, на які вже немає жодного посилання.

@2 Підсумки

@3 Об'єкт - це зібрання властивостей.
@3 Поряд з властивостями-даними є властивості-функції.
@3 Властивості можна додавати і видаляти протягом усього життя об'єкта.
@3 Збирач сміття звільняє пам'ять від непотрібних об'єктів.
