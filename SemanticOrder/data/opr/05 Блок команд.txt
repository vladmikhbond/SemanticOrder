@1 05 Блок команд

@2 Рішення квадратного рівняння
@@ -

@4 ax² + bx + c = 0

@3 Будемо шукати тільки дійсні корені.
Дискримінант рівняння: d = b² - 4ac.

@3 Рішення
{{
// дискримінант
let d = b * b - 4 * a * c;

// корені
let x1;
let x2;
if (d >= 0)
   x1 = (-b + d ** 0.5) / (2 * a);
if (d >= 0)
   x2 = (-b - d ** 0.5) / (2 * a);

// відповідь
roots.innerHTML = "x1=" + x1 + "  x2=" + x2;
}}

@2 Блок
@@ блок+|блоц+

@4 {
   команда1;
   команда2;
   ...
}

@3 Виправлене рішення.
{{
if (d >= 0)
{
   x1 = (-b + d ** 0.5) / (2 * a);
   x2 = (-b - d ** 0.5) / (2 * a);
}
}}

@2 Область видимості
@@ област+ видимост+

@3 Блок створює в програмі область видимості.
Змінні, оголошені в блоці, невидимі поза ним.
{{
{
    let x = 1;
}
alert(x);     // error
}}

@3 Змінні, оголошені поза блоком, доступні в самому блоці.
{{
let y = 2;
{
    let x = y;
}
}}

@3 Оголошення змінної повинно передувати її використанню.
{{
{
    let x = y; // помилка
}
let y = 2;
}}

@2 Правила форматування коду
@@ форматуванн+ код+

@3 1.Вкладеність команд показують відступами. Відступи роблять за допомогою пробілів.

@3 2.Закриваюча дужка блоку повинна розташовуватися точно під відкриваючою.

@3 3.Якщо в програмі є кілька смислових частин, їх відділяють одну від одної порожніми рядками.


@2 Підсумки

@3 В умовну команду можна вкласти лише одну команду,
@3 Але з кількох команд можна створити одну, поєднавши їх в блок.
@3 Змінні, оголошені в блоці, недосяжні за його межами.
@3 Код потребує форматування.
