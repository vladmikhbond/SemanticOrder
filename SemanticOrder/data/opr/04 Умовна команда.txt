@1 04 Умовна команда

@2 Умовна команда
@@ умовн+ команд+|if|умов+

@3 Команди в програмі виконуються послідовно, в тому порядку, в якому написані.
{{
alert("Число X додатне");
alert("Число X від'ємне");
alert("Число X дорівнює нулю");
}}

@3 {Зауваження}. Крапка з комою є роздільником команд в JS.
Якщо роздільник відсутній, то майже завжди інтерпретатор здатний сам його відновити.
Наприклад, інтерпретатор не ставить ";" перед квадратними дужками.

@3 На виконання команди можна накласти умову.
{{
let x = 5;

if (x > 0) alert("Число X додатне");
if (x < 0) alert("Число X від'ємне");
if (x == 0) alert("Число X дорівнює нулю");
}}

@3 Комбінацію умови і команди називають {умовною командою}.

@4 if (умова)
    команда

@3 Спочатку перевіряється умова. Якщо умова правдива, то виконується вкладена команда.
Якщо умова неправдива, вкладена команда ігнорується.

@3 Свідомі програмісти кожну команду пишуть в окремому рядку, а вкладеність показують відступом.
{{
if (x > 0)
    alert("Число X додатне");
if (x < 0)
    alert("Число X від'ємне");
if (x == 0)
    alert("Число X дорівнює нулю");
}}

@3 Приклад. Знайти корінь рівняння ax + b = 0.
{{
let x;
if (a != 0)
   x = -b / a;
alert(x);
}}

@3 Вкладеною може бути будь-яка команда, в тому числі і умовна.
Перевіримо припущення, що х лежить між -1 і 1.
{{
let x = ???
if (x > -1)
   if (x < 1)
      alert("- 1 < x < 1");
}}


@2 Логічні вирази
@@ true|false|логічн+ вираз+

@3 Вираз, значенням якого є число, зветься арифметичним.
Вираз, який може бути істинним або хибним, зветься логічним.

@3 У логічних виразів є лише два можливих значення: true або false.
{{
alert(3 < 5);   // true
}}

@3 Умова в умовній команді задається логічним виразом.

@3 Порівняння - логічні вирази. В JS їх шість.
{{
a == b
a != b
a > b
a < b
a <= b
a >= b
}}

@@ {{

@@      alert(a);
@@ if (a <= b)

@@ }}
      
@2 Двостороння умовна команда
@@ else

@4if (умова)
    команда
else
    команда

@3 Інше рішення попередньої задачі.
{{
if (a > b)
    alert(a);
else
    alert(b);
}}

@2 Схеми алгоритмів

@3 Щоб не заплутатися в складних випадках, малюють схему алгоритму.
Команди на схемах позначають прямокутниками, а умови - ромбами.

@3 Приклад 3. У змінних a, b, c  знаходяться числа. Показати у вікні alert більше з них.

@2 Підсумки

@3 Умовна команда розгалужує обчислення.
@3 В JS існують одностороння і двостороння умовні команди.
@3 Поряд з арифметичними існують логічні вирази зі значеннями true і false.
