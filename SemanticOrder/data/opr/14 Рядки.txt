@2 Екранування символів
@@ екран+ символ+|\n|\t|\r

@3 Вважається, що кожен символ має два значення - основне і альтернативне.
Екран змінює значення наступного за ним символу з основного на альтернативне.
В JS екрануючим символом є '\'.

@3 
{Символи        Основне                        Aльтернативне}
  '   "                 обмежувати рядок      позначати себе
  \                     екранувати                   позначати себе
  n t r               позначати себе             бути керуючими
  інші               позначати себе            позначати себе

@2 Операції з рядками
@
@3 Рядки можна присвоювати і порівнювати.
Порівняння рядків - лексикографічне, тобто той рядок менше, який за алфавітом стоїть раніше.

@3 Рядки можна з'єднувати операцією {+}.
Інші операції до рядків не застосовуються.

@3 Рядки - незмінні значення.
Вирази s[0], s[1], s[2] дозволяють звернутися до нульового, першого і подальших символів рядка.
Але їх не можна ставити в ліву частину оператора присвоювання, тому що рядки - незмінні значення.

@3 Не існує ніякого способу змінити рядок
(як, втім, не існує способу змінити і число).

@2 Кодування символів
@@ код+ символ+|charCodeAt|fromCharCode

@3 Символи представлені в пам'яті числами - своїми кодами.

@3 Перейти від символу до його коду дозволяє метод рядка charCodeAt()
{{
"abc".charCodeAt(0)       // 97
}}

@3 Зворотний перехід
{{
String.fromCharCode(97)   // a
}}

@3 Приклад -  шифр ​​Цезаря.
Визначити функцію ceasar(line, k), яка отримує рядок line і зміщення k і повертає зашифрований рядок.
{{
function ceasar(line, k = 1) {
    let res = "";
    for (let i = 0; i < line.length; i++) {
        let j = line.charCodeAt(i);
        let c = String.fromCharCode(j + k);
        res += c;
    }
    return res;
}
}}

@2 Unicode

@3 Існує багато способів закодувати символи числами.
Кодування бувають різні, в JS використовується кодування Unicode у форматі UTF-8.
{{

1110xxxx 10xxxxxx 10xxxxxx
11110xxx 10xxxxxx 10xxxxxx 10xxxxxx
}}

@3 В рядкові літерали також можна вставляти шістнадцятькові коди символів.
{{
alert("\u2588\u2593\u2592\u2591")
}}
