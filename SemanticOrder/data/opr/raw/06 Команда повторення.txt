@1 06 Команда повторення

@3 Щоб виконати команду з умовою, її потрібно вкласти в команду if  
(вкладена команда виконується 0 або 1 раз).
Щоб виконати команду багаторазово, її потрібно вкласти в команду while 
(вкладена команда виконується 0 або n разів).

@2 Цикл while

@4while (умова)
   команда;

@3 Перевіряється умова. Якщо вона правдива, то виконується вкладена команда.
Знову перевіряється умова і т. д.
Коли умова стає неправдивою, команда while завершується.

@3 Вивести на консоль числа від 0 до 9.
{{
let i = 0;
while (i < 10)
{
   console.log (i);
   i = i + 1;
}
}}

@3 Нескінченне повторення.
{{
while (true)
    ...
}}
 
@3 Жодного повторення.
{{
while (false)
    ...
}}

@3 {Приклад}. Додавання дробів | Скласти числа 1 + 1/2 +1/3 + ... + 1/n.

@2 Комбіновані присвоювання

@4 змінна += вираз;

@3 -=, *=, /=,%=, ...

@2 Масовість алгоритмів

@3 {Масовість} означає, що алгоритм можна застосувати до потенційно нескінченної множини вхідних даних.
Масовість досягається завдяки параметрам.
Питання. Що є параметром в задачі складання дробів?


@2 Підсумки

@3 Присвоювання, умовна команда и цикл достатні, щоб запрограмувати будь-який алгоритм.

@3 Команда, вкладена в цикл, повинна змінювати умову повторення.

@3 Масовість - невід'ємна властивість алгоритмів.

@3 Масовість забезпечують параметри.
