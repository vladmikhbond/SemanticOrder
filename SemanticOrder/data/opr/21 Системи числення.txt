@2 Переклад числа з двійкової системи в десяткову

@3 Двійкове число задано у вигляді рядка з одиниць і нулів.

@3 {Алгоритм}
Отримуємо всі складові формули по черзі, проходячи рядок справа наліво.
{{
"1011" = 1 * 1 + 1 * 2 + 0 * 4 +1 * 8 = 11
}}

@3 Алгоритм легко узагальнити на b-ічну систему числення.

@3 Приклад. Реалізація алгоритму для b <= 10.
{{
function fromString(a, b)
{
    let n = 0;
    let w = 1;
    for (let i = a.length - 1; i >= 0; i--) {
        n += w * a[i];
        w *= b;
    }
    return n;
}
}}

@2 Переклад десяткового числа в двійкову систему
@@ -

@3 Приклад. 11 -> ?
{{
11 \ 1
 5 \ 1
 2 \ 0
 1 \ 1
 0 \
}}

@3 Алгоритм.
Розділити число на 2, отримавши неповну частку і залишок.
Залишок зробити молодшим розрядом майбутнього числа, 
    а над часткою продовжити роботу.
Коли частка дорівняється нулю, зупинитися.

@3 Алгоритм легко узагальнити на b-ічну систему числення.
{{
function toString(n, b) {
    let str = "";
    while (n) {
        let r = n % b;
        n = (n - r) / b;
        str = r + str;
    }
    return str;
}
}}
