@1 43 ФП Типи даних

@2 Користувацькі типи

@3 В Гаскелі спостерігається аналогія між даними і типами.
Є конструктори даних и конструктори типів.
Iснують типи-суми і типи-добутки.
Бувають типи типів, вони звуться сорти (kinds).
Компілятор вирішує систему рівнянь з типами, коли самостійно виводить тип.

@3 Приклад типу-суми.
{{
data Color = Red | Green | Blue   
}}

@3 Компілятор здатен самостійно надати користувацькому типу деякий API.
{{
data Color = Red | Green | Blue   
  deriving (Show, Eq, Ord, Read)
}}

@3 Приклад типу-добутку
{{
data Point = Point Float Float 
    deriving Show
}}

@3 Приклад узагальненого типу
{{
data Point a = Point a a
    deriving Show
}}

@3 Приклад рекурсивного типу
{{
data Tree a = L | T a (Tree a) (Tree a) 
    deriving (Eq, Show)
}}

@2 Класи типів

@3 Eq, Show, Ord, Eq та інш. - це назви стандартних класів типів (а по-імперативному, програмних інтерфейсів).
