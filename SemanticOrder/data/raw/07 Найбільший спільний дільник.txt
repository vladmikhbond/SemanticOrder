@1 07 Найбільший спільний дільник

@3 {Завдання}.  Є два числа, наприклад, 12 і 8.
Знайти їх найбільший спільний дільник (НСД).

@3 {Рішення перше}
Дільники числа 12: 2, 3, 4, 6, 12.
Дільники числа 8: 2, 4, 8.
Спільні дільники: 2, 4.
Найбільший спільний дільник: 4.

@2 Алгоритм Евкліда

@3 [[euklide.png]]
Евклід займався геометрією, тому шукав НСД не чисел, а відрізків.
НСД двох відрізків - це найбільший відрізок, який ціле число разів укладається в обох відрізках.
[[euclide_alg.png]]


@3 {Рішення друге}
Якщо якийсь відрізок x цілу кількість разів вкладається у відрізки a і b,
то він вкладеться також і в остачу - відрізок довжиною a % b.
Тобто, шукати НСД(a, b) - теж саме, що шукати НСД(b, a % b).
НСД(a, b) = НСД(b, a % b)

@3 Приклади:
{{
НСД(12, 8) = НСД(8, 4) = НСД(4, 0)
НСД(35, 10) = НСД(10, 5) = НСД(5, 0)
}}

@3 Якщо остача дорівнює нулю, то менший з відрізків і є найбільша загальна міра.
{{
1) НСД(4, 0) = 4
2) НСД(5, 0) = 5
}}

@3 Програма
{{
let a = 35;
let b = 10;

while (b > 0)
{
    let r = a % b;
    // Призначаємо менше число більшим.
    a = b;
    // Призначаємо залишок меншим числом.
    b = r;
}
console.log (a);
}}

@2 Фінітність алгоритмів

@3 Для цілих чисел алгоритм Евкліда завжди завершується, для дійсних - не завжди.

@3 Для сторони і основи цього трикутника немає загальної міри.
[[Degrees36.png]]
Алгоритм Евкліда для нього ніколи не завершиться.

@3 Відношення сторони до основи цього трикутника не можна виразити у вигляді дробу a / b
(інакше, у них була б загальна міра - відрізок довжини НСД(a, b) ).
Отже, Евклід своїм алгоритмом и цим трикутником доводить існування ірраціональних чисел.

@3 Здатність обчислювати результат за кінцеве число кроків називається {фінітністю} алгоритму.

@2 Підсумки

@3 Евклід своїм алгоритмом и цим трикутником доводить існування ірраціональних чисел.

@3 Алгоритм Евкліда завжди завершується тільки для раціональних чисел.

@3 Програмісти можуть заробляти тільки на фінітних алгоритмах,
тобто таких, що дають результат за кінцеве число кроків.
